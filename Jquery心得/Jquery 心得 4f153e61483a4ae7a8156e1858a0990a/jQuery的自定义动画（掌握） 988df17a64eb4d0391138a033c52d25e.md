# jQuery的自定义动画（掌握）

今天看了南江老师讲的自定义动画，感觉直接颠覆我对自定义动画的认知，

在学jquery自学期间的时候，我接触的最多的就是jquery，今天补充了此知识点的知识盲区

首先animate有4个参数

第一个参数：属性

第二个参数：动画时间

第三个参数：动画速度，字符串“”,linear，和swing，默认是swing，是减速->加速->减速的动画效果

第四个参数：回调函数

第一种动画方式：普通方式

第二种动画方式：叠加方式{

在属性的值中这样写：{

width：“+=100”

}

这样就是没过一次动画，宽在原来的基础上+100px

}

第三种动画方式：关键字方式{

在属性值中这样写：{

width：”hide//toggle等等”

}

这样就是通过关键字来设置宽的样式

}
