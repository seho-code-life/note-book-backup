变量名规范

1.严重声明 ：不能用中文为变量名

2.第一字符可以用字母 _ $ (不能用数字)（有意义的标签也不可以）

3.其他字符可以用下划线 数字 字母 和 $

4.对于js中的一些关键字 还有保留词 也不能用

5.严格区分大小写·

非标准规范

1.见名知意

2.驼峰命名法

alert(); 提示框

confirm(); 判断框

prompt(); 文本输入弹框

document.getElementById("box");
